## Features

<CardGrid>
	<Card title="Easy experiment creation" icon="pencil">
		Create experiments with just a few configuration files
	</Card>
	<Card title="jsPsych + Go" icon="laptop">
		jsPsych frontend interacts with the server using WebSockets (leveraging [Go](https://go.dev/) and [Gorilla](https://gorilla.github.io/))
	</Card>
	<Card title="No databases" icon="document">
		All configuration and state is saved to text files and the `jones` binary comes with an HTTP server for static files (JS, CSS)
	</Card>
	<Card title="Simple deployment" icon="setting">
		Just build the project and transfer the binary (along with a few folders) to the server
	</Card>
</CardGrid>
