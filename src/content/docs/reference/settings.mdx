import { Badge } from '@astrojs/starlight/components';

## Fields

### `kind`
**type:** `string`, **values:** `'image' | 'sound' | 'video'`

The kind of the stimuli to be presented. 

### `nInterval`
**type:** `number`, **values:** `1 | 2`

Whether the experiment should be 1-interval or 2-interval.

### `fileExtension`
**type:** `string`

File extension for the stimuli. Most image, sound and video file extensions should work but *this has not been tested exhaustively*. Preference should be for common types like `jpg`, `png`, `wav` and `mp4`.

### `trialsPerBlock`
**type:** `number`

Number of trials in each block.

### `blocksPerXp`
**type:** `number`

Number of blocks in the experiment.

**type:** `boolean`


### `allowCreate`
